The Product

A reusable Next.js starter template with auth, database, background jobs, error monitoring, analytics, and a design system. All wired up. No ceremony. Clone it, add your tables, ship your product.

Operating Principles

Opinionated Defaults

Every tool has been chosen, configured, and tested together. You inherit decisions that took longer to make than the code took to write.

Serious Scaffolding

Supabase Auth with OTP. Drizzle ORM with migrations. Sentry with session replay. pg-boss for background jobs. Not a toy.

Minimal Ceremony

No monorepo. No microservices. No abstraction layers you have to learn before you can build. Two processes. One database. Ship.
